Release 說明 (v0.8.2)

新功能與優化
    • 編輯器 UX 大改版：
        - 新增平滑播放游標 (30ms 更新率)。
        - 鋼琴鍵盤 (PianoDisplay) 與捲軸 (NoteRoll) 支援播放時的動態發光回饋。
        - 播放按鈕狀態回饋 (播放時顯示紅色停止鈕)。
    • FF14 鍵位更新：
        - 新增 "FF14 37-Key" 方案 (C3-C6 Diatonic)，取代舊版 32 鍵方案。
        - 支援自然音階 (1-7, Q-U, A-K) 與 Ctrl 升降號。
    • MIDI 匯出增強：支援 Type 1 多軌 MIDI 檔案匯出 (保留音軌分離與頻道資訊)。
    • 效能優化：大幅提升編輯器在多音符專案下的索引效能 (O(N²) → O(N))。
    • 穩定性修復：修正快速拖曳或播放時可能導致的崩潰問題。
    • 多語言支援 (I18N)：
        - 新增繁體中文、簡體中文、日文、韓文與英文介面切換。
        - 側邊欄新增語言選擇按鈕。

Release Notes (v0.8.2)

New Features & Enhancements
    • Editor UX Overhaul:
        - Smooth playback cursor (30ms refresh rate).
        - Dynamic active note glow in PianoDisplay and NoteRoll during playback.
        - Play button state feedback (Shows red "Stop" button when playing).
    • FF14 Key Mapping Update:
        - Added "FF14 37-Key" scheme (C3-C6 Diatonic), replacing the old 32-key scheme.
        - Supports natural scale (1-7, Q-U, A-K) with Ctrl-modifier for accidentals.
    • Enhanced MIDI Export: Support for Type 1 multi-track MIDI export (preserves track separation and channel info).
    • Performance Optimization: Significantly improved indexing performance for projects with many notes (O(N²) → O(N)).
    • Stability Fixes: Fixed crashes that could occur during rapid scrubbing or playback.
    • Multi-Language Support (I18N):
        - Added interface support for Traditional Chinese, Simplified Chinese, Japanese, Korean, and English.
        - Added language selector to the sidebar.

Installation Note
    Please use 7-Zip to extract the archive.
    Windows built-in extraction tools may not correctly handle Unicode paths, causing the application to fail on startup.
