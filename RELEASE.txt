Release 說明 (v0.8.0)

新功能與優化
    • 編輯器 UX 大改版：
        - 新增平滑播放游標 (30ms 更新率)。
        - 鋼琴鍵盤 (PianoDisplay) 與捲軸 (NoteRoll) 支援播放時的動態發光回饋。
        - 播放按鈕狀態回饋 (播放時顯示紅色停止鈕)。
    • MIDI 匯出增強：支援 Type 1 多軌 MIDI 檔案匯出 (保留音軌分離與頻道資訊)。
    • 效能優化：大幅提升編輯器在多音符專案下的索引效能 (O(N²) → O(N))。
    • 穩定性修復：修正快速拖曳或播放時可能導致的崩潰問題。

安裝須知
    請使用 7-Zip 解壓縮。
    Windows 內建解壓工具可能無法正確處理 Unicode 路徑，導致程式啟動失敗。
